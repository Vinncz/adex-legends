GAME’S FUTURE DEVELOPMENT?

The Public Beta Environment, or PBE for short, is looking for intrepid Legends like yourself to test cutting edge ADEX LEGENDS content. So what is a PBE? Well, it's a server where you can try out upcoming features that aren't (and may never be) implemented in-game. Interested? Please consider visiting the Learn More page.
